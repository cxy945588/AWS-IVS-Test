🎥 AWS IVS Real-Time 直播測試

這是一個 Amazon IVS Real-Time Streaming 的測試環境，支援 超低延遲 (<500ms) 的即時直播。

📹 主播端 👀 觀眾端

⚠️ 開始測試前請先準備：

  1. 在 AWS Console 建立 IVS Real-Time Stage
  2. 使用 AWS CLI 生成 Participant Token：
    aws ivs-realtime create-participant-token
  3. 主播需要 PUBLISH,SUBSCRIBE 權限
  4. 觀眾只需要 SUBSCRIBE 權限

📖 詳細說明請參考 README.md

💡 提示：

  • 建議使用 Chrome 或 Edge 瀏覽器
  • 需要允許攝像頭和麥克風權限（僅主播端）
  • 可以開啟多個觀眾分頁測試多人觀看
  • GitHub Pages 已提供 HTTPS，無需額外設定
